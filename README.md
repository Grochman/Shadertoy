# Shadertoy
Minimalistic environment using OpenGL, GLEW, and GLFW for GLSL shader art development. 

### Features
- #include directive support for glsl files

### Original shaders
- Mandelbrot set
- Julia set
- Heart beat
- Black - white hole

### Resources
- Basic framework based on https://www.youtube.com/@TheCherno
